# liniarRegression.py

## Description
Linear regression implementation from scratch using gradient descent optimization.

## Main Features
- **Synthetic Data Generation**: Creates linear dataset with noise
- **Gradient Descent**: Custom implementation for parameter optimization
- **Model Evaluation**: Calculates MSE and R-squared metrics
- **From Scratch**: No sklearn dependency, pure NumPy implementation

## Key Functions
- `predict()`: Linear prediction using θ parameters
- `gradient_descent()`: Iterative parameter optimization
- `mean_squared_error()`: Model performance evaluation
- `r_squared()`: Coefficient of determination calculation

## Mathematical Foundation
Implements the linear model: y = θ₀ + θ₁x with gradient-based parameter learning.
