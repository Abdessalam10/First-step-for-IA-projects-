# hypothesisTesting.py

## Description
Statistical hypothesis testing implementation with confidence intervals and t-tests.

## Main Features
- **Confidence Intervals**: Calculates 95% confidence interval for sample mean
- **Statistical Measures**: Computes mean, standard deviation, and z-scores
- **Independent t-test**: Compares two groups using t-test statistics
- **Hypothesis Testing**: Determines statistical significance at Î± = 0.05

## Key Components
- Manual calculation of statistical parameters
- Two-sample t-test implementation
- P-value interpretation for hypothesis rejection/acceptance
