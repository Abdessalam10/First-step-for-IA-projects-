# integral&SGD.py

## Description
Mathematical integration and Stochastic Gradient Descent (SGD) implementation.

## Main Features
- **Symbolic Integration**: Uses SymPy for indefinite and definite integrals
- **Exponential Function**: Analyzes f(x) = e^(-x) integration
- **SGD Algorithm**: Implements stochastic gradient descent for linear regression
- **Synthetic Data**: Generates random dataset for optimization testing

## Key Components
- Indefinite integral computation
- Definite integral from 0 to infinity
- SGD parameter update rule: θ = θ - η * [2 * xi.T * (xi * θ - yi)]
- Random sampling for stochastic optimization
