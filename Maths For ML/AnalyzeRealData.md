# AnalyzeRealData.py

## Description
Exploratory Data Analysis (EDA) and statistical analysis on restaurant tips dataset.

## Main Features
- **Data Source**: Seaborn tips dataset (tips.csv)
- **Linear Regression**: Analyzes relationship between total bill and tip amount
- **Statistical Testing**: Performs t-test to compare tip amounts between genders
- **Visualizations**: 
  - Scatter plot with regression line
  - Distribution histogram
  - Correlation heatmap

## Key Outputs
- Regression coefficients and R-squared score
- Hypothesis test results for gender differences in tipping
- Data distribution and correlation insights
